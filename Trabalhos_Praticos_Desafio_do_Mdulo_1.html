Desafio do Módulo 1

Objetivos
Exercitar os seguintes conceitos trabalhados no Módulo:

* Uso do Power BI.
* SQL.

Enunciado
Para este desafio temos dois tipos de questões diferentes: Algumas sobre o Stack Overflow
Annual Survey para serem resolvidas com o Power BI, e outras para serem resolvidas com
consultas SQL.
1. Para as questões de relativas aos dados do Stack Overflow Survey, instale o Power
BI Desktop, baixe o dataset Stack Overflow Annual Developer Survey - 2020 e
carregue os dados no Power BI. É um dataset com diversas colunas, para nossa
atividade usaremos as seguintes: Age, Age1stCode, ConvertedComp, Country,
EdLevel, JobSat, OpSys e YearsCode.


2. Realizar consultas em SQL é algo muito comum na rotina de um profissional que
trabalha com dados. Então vamos exercitar um pouco dessa habilidade em algumas
questões. Para evitar a necessidade de instalação um servidor de banco de dados,
acesse o site SQL Online IDE, vamos utilizar o simulador do PostgreSQL. No anexo
deste desafio, existe um script em SQL que deve ser executado (basta copiar, colar
e clicar em “Run”) para criar as tabelas e popular o banco de dados.
